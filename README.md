# ocrapp

Extract text from PDFs and export to Excel — no manual data entry needed.

## Stack

- **Backend** — FastAPI (Python)
- **Frontend** — React + Vite

## Quick Start

```bash
# Backend
cd backend
./venv/bin/uvicorn app.main:app --reload

# Frontend (separate terminal)
cd frontend
npm run dev
```

Open `http://localhost:5173`.
